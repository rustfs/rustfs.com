# RustFS是国产的吗？

## RustFS是国产的吗？

是国产的，符合信创要求。并且，RustFS 是与 Amazon S3 、阿里云OSS和腾讯云COS兼容的分布式对象存储服务器。

它是完全中国自主知识产权，独立开发的，对标MinIO中间件平替的产品。

RustFS是高性能分布式对象存储器服务器，专为大型专用云基础结构而设计。使用 Amazon S3 REST API 将持久卷 (PV) 聚集到可扩展的分布式对象存储器中。

最适合用于存储非结构化数据，例如，照片、视频、日志文件、备份、VM 和容器映像。


## RustFS 是抄MinIO的吗？

MinIO是用Go语言实现的一款分布式存储,RustFS是用目前全世界最流行的Rust语言编写。

Rust这门语言的性能、安全性、难度非常之高。


## RustFS可以在Windows下运行吗？

> 可以。

RustFS可以在windows、Linux、MacOS、k8s、docker等多种裸金属、虚拟机等多种环境下运行。

并且可以在国产的麒麟、中科方德、统信、欧拉等国产操作系统下运行。

并且支持鲲鹏、海光、Intel、AMD、飞腾等多种国产CPU。
