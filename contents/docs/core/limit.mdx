# 使用限制

RustFS is a simple and fast file system library for Node.js. It is designed to be easy to use and fast. It is written in Rust and compiled to WebAssembly.

## Features

- **Simple API**: RustFS has a simple and easy-to-use API.
- **Fast**: RustFS is fast and efficient.
- **Cross-platform**: RustFS works on Windows, macOS, and Linux.
- **WebAssembly**: RustFS is written in Rust and compiled to WebAssembly.
