## 硬件选择

- Node.js 10.0.0 or later
- MacOS, Windows (including WSL), and Linux are supported
- Yarn (optional but recommended)

## Installation

To install RustFS in your project, you can use npm or yarn:

```bash
npm install rustfs
```

```bash
yarn add rustfs
```

Once installed, you can import RustFS in your project:

```javascript
import rustfs from 'rustfs';
```
